Script Execution Order Settings

How Script Execution Order Works
  • Startup Scripts: Run every time an agent connects (highest priority)
  • Autorun Scripts: Run once per agent when enabled (medium priority)
  • Manual Scripts: Run only when explicitly executed (lowest priority)
  • Order: Within each category, scripts execute in the order you set below
